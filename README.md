# Introduction
A game called `Power Napper: Escape from Lyulin`, created for the 2022 Sofia Game Jam event, part of the Global Game Jam.

# Engine
The game uses Unreal Engine 5.0.3 and it is a C++ project that mainly uses blueprints.
